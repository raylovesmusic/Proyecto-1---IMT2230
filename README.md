# Proyecto SVD IMT2230: LSA sobre Spotify Songs.
##### En este proyecto se utiliza la 'Descomposición en Valores Singulares' para explorar similitudes y particiones semánticas en las letras de canciones de Spotify que pertenecen a un mismo género musical. En el contexto de este proyecto se decidió comparar las canciones cuyo género principal es el 'rap'.

## Datos
##### Los datos utilizados provienen de un dataset de Kaggle que contiene diversa información de un conjunto de canciones de Spotify. Entre algunas columnas se consideraron las siguientes: 'track_name', 'track_srtist', 'lyrics', 'playlist_genre' y 'language'.

Fuente de los datos. https://www.kaggle.com/datasets/imuhammad/audio-features-and-lyrics-of-spotify-songs
